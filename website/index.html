Download

Ship tagged desktop releases, not ad-hoc builds.

GitHub Actions now covers CI, draft release creation, and a GitHub Pages download surface. Publish from tags, keep rollback simple, and stop hand-carrying DMGs.

Permissions

Microphone, screen recording, and accessibility are first-class support topics.

Install and recovery steps are documented so testers know what to grant, when to restart the app, and what counts as an actual capture failure.

Operations

Local telemetry and storage paths are documented before public launch.

Support issues now have a diagnostics contract: version, provider setup, session folder, telemetry database, and permission state.

Install Flow

Three steps to get from release asset to working capture session.

  1. Download the DMG. Open the latest tag in GitHub Releases and pick the build that matches your Mac.
  2. Move Talkiwi to Applications. Launch from Finder and use Open if Gatekeeper blocks the first run.
  3. Grant required permissions. Microphone, Screen Recording, and Accessibility should all be confirmed before capture troubleshooting starts.
